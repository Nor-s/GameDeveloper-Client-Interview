-   객체 풀을 이용 => 객체 생성과 삭제 오버헤드 줄임
-   코드에서 시간을 많이 잡아먹는 부분을 찾아내는 데 프로파일링이 중요
    -   성능을 발목잡지 않는 붑분을 최적화하는데 시간을 낭비 x
